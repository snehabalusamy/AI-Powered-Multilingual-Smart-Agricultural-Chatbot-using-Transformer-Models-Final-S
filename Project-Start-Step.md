Frontend Steps:
1. cd frontend
2. first do npm i command in frontend folder
3. then do npm run dev command in frontend folder
4. then open the web app in browser

Backend Steps:
1. cd backend
2. first do pip install -r requirements.txt command in backend folder
3. then do python main.py command in backend folder
4. then open the web app in browser
